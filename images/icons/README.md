# App Icons

The PWA requires two icon sizes:
- icon-192x192.png (192x192 pixels)
- icon-512x512.png (512x512 pixels)

You can create these icons by:
1. Using an online tool like https://realfavicongenerator.net/
2. Resizing the existing log.png from the images folder
3. Using image editing software to create square icons

The icons should be PNG format with transparent or solid background.
